1. {{ 'Purpose' if _en else 'Objet' }}

{% if _en %} The Anemos certificate is an attestation of avoided CO₂ emissions, issued by NEWTOWT for each delivered shipment, based on the operating data of the vessel that carried it. It is issued by NEWTOWT (a self-declared environmental claim in the meaning of EU consumer law), against a published, reproducible method — this document — and a public look-up registry ({{ site_url }}/verify). {% else %} Le certificat Anemos est une attestation d'émissions de CO₂ évitées, émise par NEWTOWT pour chaque expédition livrée, sur la base des données d'exploitation du navire qui l'a transportée. Il est émis par NEWTOWT (allégation environnementale auto-déclarée au sens du droit européen de la consommation), adossé à une méthode publiée et reproductible — le présent document — et à un registre public de vérification ({{ site_url }}/verify). {% endif %}

  • Tank-to-wake (TtW) — {{ 'on-board fuel-combustion emissions, the EU MRV perimeter (Regulation EU 2015/757).' if _en else 'émissions de combustion du carburant à bord, périmètre du règlement EU MRV (UE 2015/757).' }}
  • {{ 'Gas: CO₂ only' if _en else 'Gaz : CO₂ uniquement' }} — {{ 'excluding CH₄ and N₂O; figures are CO₂, not CO₂e.' if _en else 'hors CH₄ et N₂O ; les valeurs sont du CO₂, pas du CO₂e.' }}

4. {{ 'Formula' if _en else 'Formule' }}

CO₂ {{ 'avoided' if _en else 'évité' }} (kg) = ({{ conv_ef }} − {{ towt_ef }}) × {{ 'tonnage' if _en else 'tonnage' }} (t) × distance (km) ÷ 1 000

distance (km) = distance (NM) × 1,852

5. {{ 'Measured vs estimated data' if _en else 'Données mesurées ou théoriques' }}

{% if _en %} In order of preference: (a) measured — declared on-board fuel consumption (signed noon reports) and the actual distance sailed (GPS track); (b) estimated — the versioned NEWTOWT factor and an orthodromic distance with a routing allowance (× 1.15). The method used (« declared » / « theoretical ») is printed on every certificate, together with the distance source. An annual measured-vs-estimated reconciliation is published. {% else %} Par ordre de préférence : (a) mesuré — consommations déclarées à bord (noon reports signés) et distance réellement parcourue (trace GPS) ; (b) théorique — facteur NEWTOWT versionné et distance orthodromique avec majoration de routage (× 1,15). La méthode retenue (« declared » / « theoretical ») est imprimée sur chaque certificat, avec la source de distance. Une réconciliation annuelle mesuré/théorique est publiée. {% endif %}

6. {{ 'Verification & limits' if _en else 'Vérification et limites' }}

  • {% if _en %} Vessel emissions: monitored, reported and verified by an accredited body under EU MRV; publicly available in EMSA's THETIS-MRV registry. {% else %} Émissions des navires : surveillées, déclarées et vérifiées par un organisme accrédité dans le cadre du règlement EU MRV ; consultables dans le registre public THETIS-MRV de l'EMSA. {% endif %}
  • {% if _en %} Avoided-emissions calculation: NEWTOWT's method, published and reproducible (this document). A 2026 programme is under way for an independent review of the method and its alignment with ISO 14083:2023 / the GLEC Framework; its status is published at {{ site_url }}/preuves. {% else %} Calcul des émissions évitées : méthode NEWTOWT, publiée et reproductible (le présent document). Un programme 2026 est engagé pour une revue indépendante de la méthode et son alignement ISO 14083:2023 / GLEC Framework ; son statut est publié sur {{ site_url }}/preuves. {% endif %}
  • {% if _en %} Explicit limits: figures are CO₂ (not CO₂e); this is not a product life-cycle assessment; the conventional reference is a sector average. {% else %} Limites explicites : valeurs en CO₂ (pas en CO₂e) ; il ne s'agit pas d'une analyse de cycle de vie produit ; la référence conventionnelle est une moyenne sectorielle. {% endif %}

7. {{ 'Proper use of the figures (EU 2024/825)' if _en else 'Bon usage des chiffres (dir. (UE) 2024/825)' }}

  • {{ 'Communicate per shipment, in absolute kg of CO₂ avoided, citing the certificate reference and the verification link.' if _en else 'Communiquer par expédition, en kg absolus de CO₂ évité, en citant la référence du certificat et le lien de vérification.' }}
  • {{ 'Report under GHG Protocol Scope 3 category 4 (upstream transport) or Bilan Carbone®, on the maritime segment only.' if _en else 'Déclarer en GHG Protocol Scope 3 catégorie 4 (transport amont) ou Bilan Carbone®, sur le seul segment maritime.' }}
  • {{ 'This is a physical reduction at source: neither a credit, nor an offset, nor any form of “carbon neutrality”.' if _en else 'Il s\'agit d\'une réduction physique à la source : ni crédit, ni compensation, ni aucune forme de « neutralité carbone ».' }}
  • {{ 'Do not extend the claim to the whole product: the claim covers the maritime transport of the lot, not the product itself.' if _en else 'Ne pas étendre l\'allégation au produit entier : elle porte sur le transport maritime du lot, pas sur le produit lui-même.' }}
