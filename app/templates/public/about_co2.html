Comment {{ brand.nom_court }} calcule les émissions évitées

Méthode transparente, alignée avec le règlement européen EU MRV (UE 2015/757) et le GHG Protocol Scope 3 catégorie 4 (Upstream Transportation).

Facteurs d'émission retenus

Mode de transportg CO₂ / t.kmSource
Cargo vélique {{ brand.nom_court }} 1,5 Mesures embarquées + audit vérificateur tiers
Cargo conventionnel 13,7 IFP Energies nouvelles / IMO Fourth GHG Study

Formule appliquée

CO₂ évité (kg) = (13,7 - 1,5) x tonnage (t) x distance (km) / 1000

distance (km) = distance (NM) x 1,852

Exemple

12 palettes de vin (1,2 t) sur Le Havre → New York (3 200 NM ≈ 5 926 km) :

(13,7 - 1,5) x 1,2 x 5 926 / 1000 = 86,7 kg CO₂ évités vs porte-conteneur.

Vérification annuelle

Nos émissions effectives sont vérifiées chaque année par un organisme tiers accrédité dans le cadre du règlement EU MRV. Les rapports vérifiés sont publiés ici dès leur émission.
