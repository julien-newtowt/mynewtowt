{% if lang not in ('fr', 'en') %}{% include "public/_translation_notice.html" %}{% endif %} {# ===== SECTION 1 — D'OÙ VIENT LE CHIFFRE ? ===== #}
1 · {% if lang == 'en' %}The figure{% else %}Le chiffre{% endif %}

{% if lang == 'en' %}Where the figure comes from{% else %}D'où vient le chiffre ?{% endif %}

  • {% if lang == 'en' %}Perimeter: tank-to-wake (TtW){% else %}Périmètre : tank-to-wake (TtW){% endif %} — {% if lang == 'en' %} on-board fuel-combustion emissions. This is the perimeter of the EU MRV regulation. Inland pre- and post-carriage are not included (assumed explicitly). {% else %} les émissions de combustion du carburant à bord. C'est le périmètre du règlement EU MRV. Le pré-acheminement et le post-acheminement terrestres ne sont pas inclus (assumé explicitement). {% endif %}
  • {% if lang == 'en' %}Gas: CO₂{% else %}Gaz : CO₂{% endif %} — {% if lang == 'en' %} values are expressed in CO₂, excluding CH₄ and N₂O (these are not CO₂e). {% else %} les valeurs sont exprimées en CO₂, hors CH₄ et N₂O (ce ne sont pas des CO₂e). {% endif %}
  • {% if lang == 'en' %}Conventional reference{% else %}Référence conventionnelle{% endif %} : 13,7 g CO₂/t·km (IMO Fourth GHG Study / IFP Energies Nouvelles).
  • {% if lang == 'en' %}Our factor{% else %}Notre facteur{% endif %} : 1,5 g CO₂/t·km, {% if lang == 'en' %} derived from our vessels' real emissions, versioned (each certificate carries the factor version used). {% else %} dérivé des émissions réelles de nos navires, versionné (chaque certificat porte la version de facteur utilisée). {% endif %}

{% if lang == 'en' %}Formula{% else %}Formule{% endif %}

{% if lang == 'en' %}

CO₂ avoided (kg) = (13.7 − 1.5) × tonnage (t) × distance (km) ÷ 1 000

distance (km) = distance (NM) × 1.852

{% else %}

CO₂ évité (kg) = (13,7 − 1,5) × tonnage (t) × distance (km) ÷ 1 000

distance (km) = distance (NM) × 1,852

{% endif %}

{% if lang == 'en' %}Compatibility{% else %}Compatibilité{% endif %} : {% if lang == 'en' %} figures usable in a GHG Protocol Scope 3 cat. 4 declaration (upstream transport) and a Bilan Carbone®, bearing in mind the figure is CO₂ (TtW). {% else %} valeurs intégrables dans une déclaration GHG Protocol Scope 3 cat. 4 (transport amont) et un Bilan Carbone® — en gardant à l'esprit qu'il s'agit de CO₂ (TtW). {% endif %}

{# ===== SECTION 2 — QUI L'A VÉRIFIÉ ? ===== #}
2 · {% if lang == 'en' %}Verification{% else %}Vérification{% endif %}

{% if lang == 'en' %}Who verified it{% else %}Qui l'a vérifié ?{% endif %}

{% if lang == 'en' %} Our vessels' emissions are monitored, reported and verified by an accredited body under EU MRV (Regulation EU 2015/757). They are publicly available in EMSA's THETIS-MRV registry. {% else %} Les émissions de nos navires sont surveillées, déclarées et vérifiées par un organisme accrédité dans le cadre du règlement EU MRV (UE 2015/757). Elles sont consultables dans le registre public THETIS-MRV de l'EMSA. {% endif %}

{% if lang == 'en' %}Important distinction{% else %}Distinction importante{% endif %} : {% if lang == 'en' %} third-party verification covers our vessels' emissions. The avoided-emissions calculation (the difference with a conventional reference cargo) is our method, published and reproducible above. {% else %} la vérification tierce porte sur les émissions de nos navires. Le calcul des émissions évitées (la différence avec un cargo conventionnel de référence) est notre méthode, publiée et reproductible ci-dessus. {% endif %}

{# ===== SECTION 3 — MESURÉ OU THÉORIQUE ? ===== #}
3 · {% if lang == 'en' %}Data{% else %}Donnée{% endif %}

{% if lang == 'en' %}Measured or estimated?{% else %}Mesuré ou théorique ?{% endif %}

{% if lang == 'en' %} When real data is available, the certificate uses declared on-board fuel (signed noon reports) and the actual distance sailed; otherwise the certified factor and an orthodromic distance (× 1.15). The method used is printed on every certificate. An annual measured-vs-estimated reconciliation is published. {% else %} Quand les données réelles sont disponibles, le certificat est calculé sur les consommations déclarées à bord (noon reports signés) et la distance réellement parcourue. À défaut, il applique le facteur certifié et une distance orthodromique (× 1,15). La méthode retenue est imprimée sur chaque certificat. Une réconciliation annuelle mesuré/théorique est publiée. {% endif %}

{# ===== SECTION 4 — PUIS-JE LE RETROUVER ? ===== #}
4 · {% if lang == 'en' %}Registry{% else %}Registre{% endif %}

{% if lang == 'en' %}Can I look it up?{% else %}Puis-je le retrouver ?{% endif %}

{% if lang == 'en' %} Every certificate carries a serial number and a QR code. Scan it, or enter the reference on our public verification page: tonnage, distance, avoided CO₂, method, date and factor version — no personal data. {% else %} Chaque certificat porte un numéro de série et un QR code. Scannez-le, ou saisissez la référence sur notre page de vérification publique : vous y retrouvez le tonnage, la distance, le CO₂ évité, la méthode, la date et la version de facteur — sans donnée personnelle. {% endif %}
